### 使用方法

在线的项目展示效果可以点击链接： [React Animation Resume](https://wistyuan.github.io/resume/)

如果想在本地下载、编译、运行，可以参考以下命令：

```
git clone https://github.com/wistyuan/resume.git
cd resume/
npm install
npm run build
npm run start
```

就可以在 http://localhost:8080/index.html 上看到项目的展示效果啦
